The Crux

PWGuard is an open source, web-based password vault, built with the Play Framework, AngularJS, and other open source components. PWGuard is responsive, so that it works on both desktop browser and mobile browsers. PWGuard is small, comprised of less than 10,000 lines of code (including the SQL, HTML and CSS stylesheets). For detailed technical information, see the PWGuard GitHub repository .
